= Collectors

Collectors collect performance data via specific agents and protocols. 
This section includes the following information for each collector:

* collector-specific parameters (used in the <<performance-data-collection/collectd/configuration.adoc#ga-collectd-configuration, _collectd-configuration.xml_ file>>)
* configuration file(s)

NOTE: Understanding <<performance-data-collection/resource-types.adoc#resource-types, resource types>> helps when editing collector-specific configuration files.
